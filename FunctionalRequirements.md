# 🛒  Market (Application Of E-Commerce)

### 방향

- 기존 E-Commerce앱과 같은 기능을 가진 애플리케이션의 제작

### 구현할 기능

- 로그인 (상품 수정/삭제시 필요한 비밀번호 정보)
- 상품 등록, 상품 조회, 상품 수정, 상품 삭제 (CRUD)
- 상품 상세정보 확인
- 로컬 캐시
- *상품 키워드 검색
- *계정 관리

(*는 서버개발이후 진행예정)

### 핵심 경험

- 단위 테스트를 기반으로한 TDD
- MVVM 설계
- Auto layout (programmatically)
- 사용자 친화적인 UI/UX 고민 (HIG)
